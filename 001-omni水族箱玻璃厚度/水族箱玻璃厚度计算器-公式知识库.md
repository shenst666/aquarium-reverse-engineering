### 4.0 — 水族箱体积

**声明式公式**：`length * width * height = volume`

**物理含义**：长方体水族箱的容积，长 × 宽 × 高。

| v2 | 宽度 | `volume / (height * length)` | volume, length, height |

### 4.1 — 水族箱表面积（5面）

**声明式公式**：`(2 * height * (length + width)) + (length * width) = Area`

**物理含义**：水族箱 5 个面的总面积。前后面 + 两侧面 + 底面 = 2hl + 2hw + lw。顶面不计入（开口）。

**平面展开**：
```
         ┌─────────────┐
         │   底面 l×w   │
    ┌────┼────┬────┬────┤
    │ 前 │ 左 │ 后 │ 右 │
    │h×l │h×w │h×l │h×w │
    └────┴────┴────┴────┘
```

| 变体 | 求解目标 | 执行公式 |
|------|---------|---------|
| v0 | 高度 | `(1/2) * (Area - l*w) / (l + w)` |
| v1 | 长度 | `(Area - 2*h*w) / (2*h + w)` |
| v2 | 宽度 | `(Area - 2*h*l) / (2*h + l)` |
| v3 | **表面积** | `2*h*l + 2*h*w + l*w` |

### 4.2 — 玻璃重量

**声明式公式**：
```
glass_weight = one_way(
  glass_density / 10^9 * (
    2 * side_thickness * width * height +
    2 * side_thickness * length * height +
    bottom_thickness * length * width
  )
)
```

**物理含义**：玻璃总质量 = 玻璃密度 × 玻璃总体积。玻璃总体积 = 5块玻璃各自面积 × 各自厚度之和。前后面各1块 + 两侧面各1块 + 底面1块。

**正向公式（变体v0）**：求解 `glass_weight`
- 依赖变量：`LHratio, LWratio, bending_stress, glass_density, height, length, width`
- 注意：`side_thickness` 和 `bottom_thickness` 被内联展开，分别替换为公式16和公式18

### 4.3 — 水重量

**声明式公式**：`water_weight = volume * water_density / 10^9`

**物理含义**：水的总质量 = 水体积 × 水密度。除以 10^9 是因为 volume 是 cm³（1 cm³ = 10⁻⁹ × 10⁹ mm³... 单位换算为 kg/m³ × cm³ → kg）。

实际上完整的单位换算：
- volume: cm³ = 10⁻⁶ m³
- water_density: 998 kg/m³
- water_weight (kg) = (10⁻⁶ × volume_cm³) × 998 / (10⁻⁹ × 10⁹) ... 最终公式简化为 `volume * 998 / 1e9` 会得到 kg，但原公式用 `1/1e9` 是为了和 glass_weight 统一单位体系。

| 变体 | 求解目标 | 执行公式 |
|------|---------|---------|
| v0 | 体积 | `1e9 * water_weight / water_density` |
| v1 | 水密度 | `1e9 * water_weight / volume` |
| v2 | **水重量** | `(1/1e9) * volume * water_density` |

**物理含义**：这是平板弯曲理论中挠度系数的 4 次多项式拟合。α 决定了玻璃在水压下弯曲多少。来源于弹性力学中矩形板受均匀载荷的 Navier 解或有限元分析结果的回归拟合。

### 4.9 — 侧板真实 α 系数（分段限幅）

**声明式公式**：
```
true_alpha_coeff_side = if_else(
  LHratio < 0.5,  0.003,
  if_else(LHratio > 3, 0.067, alpha_coeff_side)
)
```

**物理含义**：4 次多项式只在 LHratio ∈ [0.5, 3] 范围内有效。超出范围时，多项式会快速发散产生物理上不合理的结果，因此做截断处理。

| LHratio 范围 | 物理场景 | α_side_true |
|-------------|---------|------------|
| < 0.5 | 极矮缸（长>>高），玻璃几乎不受弯 | **0.003** |
| 0.5 ~ 3 | 正常比例 | 多项式计算 |
| > 3 | 极高缸（高>>长），玻璃弯曲饱和 | **0.067** |

**正向公式（变体v0，三元运算符形式）**：
```
alpha_side_true = (LHratio < 1/2) 
  ? 3/1000 
  : ((LHratio > 3) 
      ? 67/1000 
      : 多项式)
```

### 4.15 — 弯曲应力

**声明式公式**：`bending_stress = glass_tensile_strength / safe_factor`

**物理含义**：许用弯曲应力 = 材料抗拉强度 / 安全系数。玻璃抗拉强度 19.2 MPa，除以安全系数 3.8，得到允许使用的最大弯曲应力约 **5.05 MPa**。

这就是为什么 19.2 MPa 的玻璃不会在 5 MPa 的水压下断裂——安全系数保证了长期使用的可靠性。

| 变体 | 求解目标 | 执行公式 |
|------|---------|---------|
| v0 | 安全系数 | `strength / bending_stress` |
| v1 | 抗拉强度 | `bending_stress * safe_factor` |
| v2 | **弯曲应力** | `strength / safe_factor` |

### 4.16 ⭐ — 侧板玻璃厚度（核心公式）

**声明式公式**：
```
side_thickness = one_way(
  sqrt(true_beta_coeff_side * height^3 * 0.00001 / bending_stress)
)
```

**物理含义**：这是整个计算器最重要的公式。来源于平板弯曲理论：

```
t = sqrt(β × h³ × ρg × 10⁻⁵ / σ_allow)

其中：
  t      = 玻璃厚度 (mm)
  β      = 弯曲应力系数（取决于 LHratio）
  h      = 水族箱高度 (cm)
  ρg     = 水的重度 ≈ 0.00001 (简化系数)
  σ_allow = 许用弯曲应力 (MPa)
```

**为什么高度是 3 次方？** 静水压力随深度线性增加（P = ρgh），水对玻璃的总力 ∝ h²（三角形分布面积），合力作用点在 h/3 处，因此弯矩 ∝ h³，厚度 ∝ √(弯矩) → t ∝ h^1.5。

**正向公式（变体v0，完整展开）**：
```
side_thickness = (1/1000) * sqrt(10) 
  * sqrt(height³ * β_side_true / bending_stress)
```

其中 `β_side_true` 内联展开为公式 10 的分段多项式。

这个公式输出单位为 **mm**（因为乘以 1/1000 且 sqrt(10) 来自单位制转换）。

### 4.17 — 侧板挠度

**声明式公式**：
```
side_deflection = one_way(
  true_alpha_coeff_side * height^5 * 9.81 * 0.000001 
  / (69000 * side_thickness^3)
)
```

**物理含义**：玻璃面板在水压下弯曲变形的最大位移量。

```
δ = α × h⁵ × ρg × 10⁻⁶ / (E × t³)

其中：
  δ      = 挠度 (mm)
  α      = 挠度系数（取决于 LHratio）
  h      = 水族箱高度 (cm)
  ρg     ≈ 9.81 × 10⁻⁶ (水的重度，含单位换算)
  E      = 69000 MPa (玻璃弹性模量)
  t      = 玻璃厚度 (mm)
```

**挠度与高度的 5 次方成正比！** 这意味着把缸加高 2 倍，玻璃弯曲量增加 32 倍。

**正向公式（变体v0）**：
```
side_deflection = (327/2300000000000) 
  * height^5 * α_side_true 
  / side_thickness^3
```

**声明式公式**：
```
bottom_thickness = one_way(
```

**物理含义**：与侧板厚度公式结构完全相同，但使用 `β_bottom_true`（基于 LWratio）。底板承受的水压分布与侧板不同，因为底板是水平放置的四边支撑板，且承受的是整个水柱的均匀压力（而非三角形分布）。

```
12. 公式9:  true_alpha_coeff_side = clamp(alpha_coeff_side, LHratio)

14. 公式11: alpha_coeff_bottom = f11(LWratio)
15. 公式13: true_alpha_coeff_bottom = clamp(alpha_coeff_bottom, LWratio)
16. 公式19: bottom_deflection = true_alpha_coeff_bottom * height^5 * 9.81e-6 / (69000 * bottom_thickness^3)
```

## 8. 工程背景知识

### 8.1 为什么玻璃厚度取决于高度而非长度？

静水压力公式：`P = ρ × g × h`

- 在同一深度，水压只取决于水深（h），与水平方向尺寸无关
- 高度越大 → 底部水压越大 → 需要更厚的玻璃
- 长度和宽度影响的是玻璃面板的面积（因此影响重量），但不影响压强

### 8.2 α 和 β 系数的物理意义

- **α 系数**：无量纲挠度系数，来自矩形板在均布/三角形分布载荷下的弹性力学解析解
  - 值越大 → 玻璃弯曲越多
- **β 系数**：无量纲应力系数
  - 值越大 → 玻璃承受的弯曲应力越大 → 需要越厚

### 8.3 安全系数 3.8 的含义

- 玻璃长期强度只有短期强度的约 25-30%（由于亚临界裂纹扩展）
- 安全系数 3.8 意味着：允许应力 = 19.2 / 3.8 ≈ 5.05 MPa
- 这是水族箱工程中常用的保守值

### 8.4 玻璃弹性模量 69000 MPa

- 这是钠钙玻璃（soda-lime glass）的标准弹性模量
- 约等于钢的 1/3（钢的 E ≈ 200000 MPa）

### 8.5 挠度的实际意义

- 挠度小于玻璃厚度的 1/2 通常被认为是安全的
- 过大的挠度虽然玻璃不断裂，但会导致：
  - 硅胶密封条疲劳
  - 视觉上可见的"鼓肚"效应
  - 长期蠕变导致密封失效

## 附录：对数学学生的说明

如果你正在阅读这个计算器的数学结构，以下是值得关注的学习点：

1. **分段多项式拟合**：公式 7-14 展示了如何用多项式逼近复杂的弹性力学解析解，再用分段函数处理边界情况
2. **量纲分析**：注意公式中 `×10⁻⁵`、`×10⁻⁶`、`÷10⁹` 等系数，它们来自 cm → m → mm 的单位换算链
3. **计算图**：engine 将 20 个公式解析为 DAG（有向无环图），自动检测依赖关系并决定计算顺序
4. **变体系统**：每个公式有多个变体（不同未知数），engine 根据用户输入了哪些值自动选择使用哪个变体
5. **one_way 标记**：某些公式只允许单向计算（因为内联展开后过于复杂），这是计算图优化的一种策略
